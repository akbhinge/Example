DATA MINING TOOLS

A Comparative Study


Overview


The main objective of this comparative study is to help to learn the working of various data mining tools. The comparative study shows the detailed implementation of classification and clustering algorithms in Weka and Rapid Miner and also provides information about the working of Tableau and R, which are some of the other data mining and statistical analysis tools.


Data Mining Tools


Various Data Mining tools exist, some open source , some proprietary and are used to mine the data contained in the form of data sets. These data sets contain information about a number of topics. Generally, the data sets are large files containing vast information which needs to be analysed. Data Mining tools help to get knowledge from these data sets to do data analysis. A number of classification and clustering algorithms can be implemented using the Data Mining tools.

A number of popular data mining tools like Weka, Rapid Miner and other tools like R-Programming integrated with Tableau provide an efficient method to implement the various data mining algorithms.

The comparative study provides information about all these tools along with a number of useful resources for your reference! All at one place!
